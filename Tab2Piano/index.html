This tool allows you to convert guitar tab notes to piano keys. You input the string name and the fret number, and it visualizes the corresponding piano key. The string names follow the format of octave + note. Octaves before the middle octave are labeled with a negative number, e.g. -1B (the first key before the middle octave), while octaves after the middle octave have positive numbers, e.g. 1C (the first key after the middle octave). The middle octave does not have a number before it, e.g. C, C#. Black keys are labeled with a # at the end, e.g. 1C# (the first black key after the middle octave).

Keybinds: You can reset the frets by pressing "Ctrl+R". You can also switch between inputs by pressing the arrow keys "Right/Left Arrow Keys" to switch to the input on the left/right, "Top/Bottom Arrow Keys" to switch to the input above/below, "+/-" (including numpad versions) and normal "=" keys to increase the value of the input by one. You can use "Ctrl+U" or "Ctrl+J" to increase/descrease the tuning of the guitar by one semitone, as well as the up/down arrow buttons.
